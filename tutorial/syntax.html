C# Syntax Regels

C# is streng. Als je een puntkomma vergeet, stopt alles met werken.

Het Skelet

// Program.cs in .NET 6+ (Modern)
Console.WriteLine("Hello World");

In oudere tutorials zie je vaak namespace en class Program. Dat hoeft niet meer voor simpele apps (Top-level statements).

Belangrijke Regels

1. Statements eindigen met ;

Elk commando moet afgesloten worden. Net als een punt in een zin.

Console.WriteLine("Dit werkt"); 
Console.WriteLine("Dit werkt niet") // Error!

2. Case-Sensitive (Hoofdlettergevoelig)

console is niet hetzelfde als Console.

console.WriteLine(); // FOUT
Console.WriteLine(); // GOED

3. Blokken met { }

Code die bij elkaar hoort (bijv. in een `if` of `loop`) staat tussen accolades. 

if (true) 
{
    // Dit hoort bij de if
}

Console.WriteLine("Hallo")

Toon Antwoord

1. Regel 1 mist een puntkomma ;
2. Regel 2 gebruikt kleine letters voor console en writeline (Moet Console.WriteLine zijn).

JSON in C#

Ontdek hoe je data uitwisselt met het razend populaire JSON formaat.

Wat is JSON?

JSON (JavaScript Object Notation) is d√© standaard voor het uitwisselen van data op het web. Het is leesbaar voor mensen en makkelijk te parsen voor computers. 

{
  "naam": "Marouane",
  "leeftijd": 25,
  "isStudent": true,
  "skills": ["C#", "HTML", "CSS"]
}

System.Text.Json

In modern C# (.NET Core en nieuwer) gebruiken we de ingebouwde library System.Text.Json. Deze is supersnel en veilig. Vergeet niet de namespace toe te voegen! 

using System.Text.Json;

Serialisatie (Object -> JSON)

Om een C# object om te zetten naar een JSON string, gebruik je JsonSerializer.Serialize. 

public class Persoon 
{
    public string Naam { get; set; }
    public int Leeftijd { get; set; }
}

Persoon p = new Persoon { Naam = "Ali", Leeftijd = 21 };
string jsonString = JsonSerializer.Serialize(p);

Console.WriteLine(jsonString); 
// Output: {"Naam":"Ali","Leeftijd":21}
üí° Tip:
Gebruik JsonSerializerOptions met WriteIndented = true voor mooi opgemaakte (leesbare) JSON!

Deserialisatie (JSON -> Object)

Om een JSON string terug om te zetten naar een C# object, gebruik je JsonSerializer.Deserialize. 

string json = "{\"Naam\":\"Sarah\", \"Leeftijd\":30}";

Persoon p = JsonSerializer.Deserialize(json);

Console.WriteLine(p.Naam); // Output: Sarah

Expert Level: Advanced JSON

Soms is de JSON "vies" (bijv. snake_case: "user_id") maar wil je nette C# Properties (PascalCase: UserId).

1. Properties Mappen

using System.Text.Json.Serialization;

public class User 
{
    // JSON is "user_id", maar C# is "UserId"
    [JsonPropertyName("user_id")]
    public int UserId { get; set; }
}

2. Hoofdletters Negeren

Standaard zijn "Naam" en "naam" NIET hetzelfde. Dit lost het op: 

var opties = new JsonSerializerOptions 
{ 
    PropertyNameCaseInsensitive = true 
};
var p = JsonSerializer.Deserialize(json, opties);

JSON & Bestanden

Vaak wil je instellingen of data opslaan in een .json bestand. Dit combineert System.IO met JSON. 

// Opslaan (Schrijven)
string json = JsonSerializer.Serialize(mijnLijst);
File.WriteAllText("data.json", json);

// Laden (Lezen)
string gelezenJson = File.ReadAllText("data.json");
var lijst = JsonSerializer.Deserialize>(gelezenJson);
üéØ Oefening: Settings Manager

Maak een programma dat onthoudt wat de favoriete kleur van de gebruiker is.
1. Maak een class Config met property FavoriteColor.
2. Bij opstarten: Probeer "config.json" te laden en pas de kleur toe (toon in console).
3. Vraag gebruiker om nieuwe kleur.
4. Sla op in "config.json".

Toon Antwoord 
class Config { public string Color { get; set; } }

// Laden
if (File.Exists("config.json")) {
    string json = File.ReadAllText("config.json");
    Config c = JsonSerializer.Deserialize(json);
    Console.WriteLine($"Vorige kleur: {c.Color}");
}

// Nieuwe invoer
Console.Write("Wat is je favoriete kleur? ");
string input = Console.ReadLine();

// Opslaan
Config nieuw = new Config { Color = input };
string opslaan = JsonSerializer.Serialize(nieuw);
File.WriteAllText("config.json", opslaan);

C# Break & Continue

Soms wil je een loop vroegtijdig stoppen of overslaan.

Breken (Break)

Stopt de loop DIRECT en gaat verder met de code eronder.

for (int i = 0; i < 10; i++) 
{
    if (i == 4) 
    {
        break; // Stop als we bij 4 zijn
    }
    Console.WriteLine(i); 
}
// Output: 0, 1, 2, 3

Overslaan (Continue)

Slaat DEZE ronde over, maar gaat door met de volgende.

for (int i = 0; i < 5; i++) 
{
    if (i == 2) 
    {
        continue; // Sla nummer 2 over
    }
    Console.WriteLine(i);
}
// Output: 0, 1, 3, 4

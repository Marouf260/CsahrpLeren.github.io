C# Break & Continue

Soms wil je een loop vroegtijdig stoppen of overslaan.

Breken (Break)

Stopt de loop DIRECT en gaat verder met de code eronder.

for (int i = 0; i < 10; i++) 
{
    if (i == 4) 
    {
        break; // Stop als we bij 4 zijn
    }
    Console.WriteLine(i); 
}
// Output: 0, 1, 2, 3

Overslaan (Continue)

Slaat DEZE ronde over, maar gaat door met de volgende.

for (int i = 0; i < 5; i++) 
{
    if (i == 2) 
    {
        continue; // Sla nummer 2 over
    }
    Console.WriteLine(i);
}
// Output: 0, 1, 3, 4
🎯  Denk-Oefening

Wanneer is dit nuttig?

Toon Antwoord

Bijvoorbeeld als je door een lijst gebruikers loopt, maar je wilt de geblokkeerde gebruikers overslaan (continue) of stoppen zodra je de juiste persoon hebt gevonden (break).

Deep Dive: Nested Loops

Let op: `break` stopt alleen de huidige loop (de binnenste). Als je een loop IN een loop hebt, en je wilt HELEMAAL stoppen, heb je hulp nodig (b.v. een `return` of een `bool found = true`).

🎯 Extra Challenge: Eerste Veelvoud

Zoek het eerste getal tussen 10 en 100 dat deelbaar is door 7 EN deelbaar is door 5. Zodra je hem hebt: print hem en STOP met zoeken (break).

Toon Antwoord 
for (int i = 10; i <= 100; i++) 
{
    if (i % 7 == 0 && i % 5 == 0) 
    {
        Console.WriteLine($"Gevonden! Het is {i}"); // 35
        break; // Stop, we hoeven niet verder te zoeken naar 70 etc.
    }
}

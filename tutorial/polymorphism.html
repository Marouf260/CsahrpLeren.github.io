C# Polymorphism

Polymorfisme betekent "vele vormen". Het laat verschillende classes op hun eigen manier reageren op dezelfde methode.

Virtual en Override

De Parent geeft toestemming (virtual) en de Child past het aan (override).

class Dier 
{
    public virtual void MaakGeluid() 
    {
        Console.WriteLine("Dier geluid...");
    }
}

class Hond : Dier 
{
    public override void MaakGeluid() 
    {
        Console.WriteLine("Woef woef!");
    }
}

class Kat : Dier 
{
    public override void MaakGeluid() 
    {
        Console.WriteLine("Miauw!");
    }
}

Waarom is dit handig?

Je kunt nu een lijst van Dieren maken en ze allemaal laten geluid maken, zonder dat je weet welk dier het precies is!

List<Dier> dierenTuin = new List<Dier>();
dierenTuin.Add(new Hond());
dierenTuin.Add(new Kat());

foreach (Dier d in dierenTuin) 
{
    d.MaakGeluid(); // De hond blaft, de kat miaauwt. Magie!
}

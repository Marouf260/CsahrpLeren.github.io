Introductie tot C#

C# (uitgesproken als "C-Sharp") is de taal van Microsoft voor alles: web, cloud, games en AI.

Wat is .NET?

Als C# de auto is, is .NET de motor en de snelweg. Het is het ecosysteem en de runtime waarin je code draait.

Wat is een Console App?

Dit is de puurste vorm van programmeren. Geen knoppen, geen muis, alleen tekst in en tekst uit. Dit is waar je de logica leert zonder afleiding van HTML/CSS.

🎯  Denk-Oefening

Waarom leren we eerst Console en geen grafische apps?

Toon Antwoord

Omdat grafische interfaces (buttons, kleuren) afleiden van de kern: data structuren, algoritmes en logica. Als je Console meestert, is een UI toevoegen makkelijk.

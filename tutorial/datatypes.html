Value vs Reference Types

Dit is cruciaal om te snappen hoe C# werkt.

Value Types (Stack)

Simpele types zoals int, double, bool. De waarde zit direct in de variabele.

Reference Types (Heap)

Complexe types zoals string, class, ⚡. De variabele is een adreskaartje naar de data.

🎯 Extra Challenge: The Overflow

Een `byte` kan maximaal 255 zijn. Wat gebeurt er als je er 1 bij optelt? Probeer het uit in code!

Toon Antwoord 
byte b = 255;
b++; // Tel er 1 bij op
Console.WriteLine(b); // Output: 0 🎯

Dit heet Overflow. De teller slaat "over de kop" terug naar 0.

🎯  Denkvraag

Waarom gebruiken we niet altijd double voor alle getallen?

Toon Antwoord

1. Performance: double is trager. 2. Precisie: double kan afrondingsfouten maken (niet goed voor geld!). 3. Geheugen: double is 2x zo groot als int.

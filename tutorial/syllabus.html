Cursusoverzicht (Syllabus)

Hier is het complete overzicht van wat je leert in deze Masterclass.

  • Deel 1: De Basis - Syntax, Variabelen, Data Types, Casting, User Input.
  • Deel 2: Logica - Operators, Math, Strings, Booleans, If/Else, Switch.
  • Deel 3: Loops & Arrays - While, For, Break/Continue, Arrays.
  • Deel 4: Methods - Parameters, Return types, Overloading, Ref/Out.
  • Deel 5: Object Oriented Programming (OOP) - Classes, Objects, Constructors, Inheritance, Polymorphism, Interfaces.
  • Deel 6: Advanced - File I/O, Exceptions (Try/Catch), Enums.
  • Deel 7: Internals - Compiler, Memory (Stack/Heap), Garbage Collection.

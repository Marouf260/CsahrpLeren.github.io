C# Access Modifiers

Beveilig je code door te zeggen wie waar bij mag (Encapsulation).

De Belangrijkste 2

Modifier Toegang
public Iedereen mag erbij.
private Alleen de class zelf. (Dit is de standaard!)

Waarom Private?

Om te voorkomen dat iemand je data kapot maakt. Je beschermt de variabelen en laat mensen er alleen bij via gecontroleerde methodes (Properties).

class BankRekening
{
    private int saldo = 1000; // Niemand mag dit zomaar aanpassen!

    public void NeemOp(int bedrag) 
    {
        if (bedrag <= saldo) saldo -= bedrag;
    }
}

Andere Modifiers

  • protected: Alleen deze class en zijn kinderen (erfenis).
  • internal: Alles binnen dezelfde Applicatie (.dll).

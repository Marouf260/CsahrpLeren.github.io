C# Math

C# heeft een ingebouwde Math class voor slim rekenwerk.

Veelgebruikte Functies

Console.WriteLine(Math.Max(5, 10));  // 10 (De hoogste)
Console.WriteLine(Math.Min(5, 10));  // 5 (De laagste)
Console.WriteLine(Math.Sqrt(64));    // 8 (Wortel)
Console.WriteLine(Math.Abs(-50));    // 50 (Absoluut / positief)
Console.WriteLine(Math.Round(9.99)); // 10 (Afronden)

Random Numbers

Voor games en simulaties heb je willekeurige getallen nodig.

Random rng = new Random();
int dobbelsteen = rng.Next(1, 7); // 1 t/m 6 (7 doet niet mee!)
Console.WriteLine(dobbelsteen);
🎯  Oefening

Maak een programma dat een muntje opgooit ("Kop" of "Munt").

Toon Antwoord 
Random rng = new Random();
int worp = rng.Next(0, 2); // 0 of 1

if (worp == 0)
    Console.WriteLine("Kop");
else
    Console.WriteLine("Munt");

Deep Dive: Floating Point Errors

Computers zijn slecht in kommagetallen. `0.1 + 0.2` is soms `0.30000000004` in plaats van 0.3. Gebruik daarom `Math.Round()` als je het netjes wilt tonen, of gebruik `decimal` voor geld.

🎯 Extra Challenge: Pythagoras

Bereken de schuine zijde van een driehoek.
A = 3, B = 4.
Formule: Wortel van (A*A + B*B). Tip: Gebruik Math.Sqrt()

Toon Antwoord 
double a = 3;
double b = 4;
double schuineZijde = Math.Sqrt(a * a + b * b);

Console.WriteLine(schuineZijde); // 5

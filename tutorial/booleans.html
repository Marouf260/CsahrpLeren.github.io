C# Booleans

In programmeren is alles uiteindelijk waar (True) of niet waar (False).

De basis

Een bool kan maar twee waarden hebben. Dit is de basis van elke beslissing die een computer neemt.

bool isCsharpLeuk = true;
bool isVissenLeuk = false;

Console.WriteLine(isCsharpLeuk); // True

Boolean Expressies

Je krijgt vaak een boolean terug als je iets vergelijkt.

int x = 10;
int y = 9;

Console.WriteLine(x > y); // True (want 10 is groter dan 9)

Real World Voorbeeld

In professionele apps zie je dit vaak bij configuraties ("Flags").

bool hasAdminRights = false;
bool isDeleted = true;

if (!hasAdminRights) // Het uitroepteken betekent NIET
{
    Console.WriteLine("Geen toegang!");
}

11.1 Refactoring & Principes

Laat de camping schoner achter dan je hem vond. (Boy Scout Rule).

Code Smells

  • Magic Numbers: `if (status == 2)` -> Wat is 2? Gebruik een Enum! (`Status.Pending`).
  • God Class: Een class van 2000 regels die alles doet. Hak hem in stukken!
  • Long Parameter List: `void Doe(int a, string b, bool c, int d, ...)`. Maak een object `DoeOptions`.

DRY (Don't Repeat Yourself)

Heb je code gekopieerd? Fout. Maak een functie.

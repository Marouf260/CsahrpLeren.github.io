3.2 Loops & Iteration

Dingen herhalen is wat computers het beste kunnen.

For vs Foreach

Vroeger was for sneller. Tegenwoordig is foreach op Arrays (`[]`) door de compiler zo geoptimaliseerd dat het bijna even snel is.

Gebruik foreach voor leesbaarheid, tenzij je de index i echt nodig hebt.

Performance Hack: `Span`

Als je high-performance code schrijft, gebruik je geen Lists, maar Spans. Een Span<T> is een "venster" op geheugen.

int[] numbers = { 1, 2, 3, 4, 5, 6 };

// We willen alleen de middelste 2.
// OUD: numbers.Skip(2).Take(2) -> Maakt nieuwe allocaties!
// NIEUW: Span slice -> 0 allocaties. Supersnel.
Span slice = numbers.AsSpan().Slice(2, 2);

2.4 Boxing & Unboxing

Hoe je per ongeluk je performance om zeep helpt.

Wat is Boxing?

Boxing gebeurt wanneer je een Value Type (zoals int) in een Reference Type (zoals object of interface) stopt.

int i = 123;      // Stack (snel)
object o = i;     // BOXING! (traag)

Wat gebeurt er onder water?

  1. Er wordt een nieuw object gealloceerd op de Heap.
  2. De waarde (123) wordt gekopieerd van Stack naar dit nieuwe Heap-object.
  3. De Garbage Collector moet dit later opruimen.

Wat is Unboxing?

Het omgekeerde proces. Expliciete cast terug naar de waarde.

int j = (int)o;   // UNBOXING!

Dit checkt eerst "Is o wel echt een int?" en kopieert dan de waarde terug naar de Stack.

De grote verwarring: .NET Framework vs Core vs 5+

Er was ooit chaos. Hier is de historie in 3 regels:

  • .NET Framework (4.8 en ouder): Oude Windows-only versie. Legacy. Niet meer leren.
  • .NET Core (1.0 - 3.1): De herschrijving. Cross-platform (Linux/Mac). Open Source.
  • .NET 5, 6, 7, 8+: De samensmelting. "Core" is uit de naam gevallen. Dit is de toekomst.
‚ùå Valkuil:
Als je online zoekt, vind je vaak code voor ".NET Framework". Pas op met oudere StackOverflow antwoorden (pre-2019). Classes zoals `WebClient` zijn verouderd; gebruik `HttpClient`.

Standard Libraries (BCL)

C# komt met een gigantische bibliotheek: de Base Class Library (BCL). Dit zijn duizenden classes die Microsoft al voor je heeft geschreven (`System.Console`, `System.IO.File`, `System.Linq`).

Wij gaan deze libraries niet alleen gebruiken, we gaan snappen hoe ze werken.

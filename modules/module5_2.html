Static vs Dynamic Polymorphism

1. Static (Compile Time) - Overloading

De compiler kiest welke methode hij pakt op basis van parameters.

void Print(int i) { ... }
void Print(string s) { ... } // Zelfde naam, ander contract

2. Dynamic (Runtime) - Overriding

De runtime (CLR) kijkt in de Virtual Method Table (vtable) welk type het object echt is. 

public virtual void Speak() => Console.WriteLine("...");
public override void Speak() => Console.WriteLine("Woof!");

Virtual methods zijn iets langzamer dan normale methods omdat de CPU een 'lookup' moet doen (indirection). Meestal verwaarloosbaar, maar goed om te weten.

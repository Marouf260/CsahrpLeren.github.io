Project 16: Go West

De ultieme test voor je OOP skills. Implementeer een complete game engine.

De Missie

Je bouwt een spel dat volledig via tekst werkt. De speler moet navigeren door Rooms (Noord, Oost, Zuid, West), Items verzamelen in een Backpack, en deze items gebruiken om puzzels op te lossen.

🎯 Architectuur Tips

  • OOP: Denk in objecten. Een `Room` is een object. Een `Item` is een object.
  • Relationships: Een `Room` heeft 4 eigenschappen (N,E,S,W) die ook `Room` objecten zijn. Dit is hoe je ze aan elkaar linkt.
  • Inventory: De `Player` heeft een `List<Item>` (rugzak).

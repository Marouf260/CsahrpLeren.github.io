Project 17: Bank Systeem

Object Oriented Programming (OOP) in de praktijk.

🎯 De Uitdaging

Simuleer een bank. Je moet rekeningen kunnen aanmaken, geld storten, opnemen en overmaken.

📘 Onderzoekstips

  • Gebruik een Class `Rekening` als blauwdruk.
  • Encapsulation: Zorg dat het Saldo niet zomaar direct aangepast kan worden (`private set`), maar alleen via methodes `Storten()` en `Opnemen()`.
  • Methods: Hoe zorg je dat `Opnemen` faalt als het saldo te laag is?

Project 24: Json Cleaner

Fix lelijke data van legacy systemen.

ðŸŽ¯ De Uitdaging

Je krijgt een JSON bestand met vreselijke namen, bijv: {"usr_nm": "Piet", "bd_yr": 1990}.

Jouw taak: Lees dit in naar een nette C# class User met properties UserName en BirthYear en sla het weer op als "schone" JSON.

ðŸ“˜ Attributes

  • Gebruik [JsonPropertyName("usr_nm")] boven je C# property.
  • Dit vertelt de serializer welke vreemde naam bij welke nette property hoort.

ðŸŽ¯ Structuur Hint

using System.Text.Json.Serialization;

class User 
{
    [JsonPropertyName("usr_nm")]
    public string UserName { get; set; }

    [JsonPropertyName("bd_yr")]
    public int BirthYear { get; set; }
}

// Lees "dirty.json" -> Deserialize -> Serialize -> Sla op als "clean.json"
